Automated Alarm System

Overview

This project upgrades a basic alarm system into a sophisticated security solution using a Raspberry Pi. The enhanced system features advanced alert capabilities, a customizable user experience, and improved functionality for monitoring and managing home security.

Project Photos

Project Story

The original alarm system was limited to indicating which door was open by lighting up corresponding LEDs. To enhance its capabilities, I developed a system that integrates a Raspberry Pi to monitor and respond to various events. The initial version involved soldering wires to the alert lights, connecting them to a Raspberry Pi, which measured voltages to detect open doors and low battery conditions through LED flash patterns. This system utilized a simple LCD screen to display modes and a button to switch between the modes. The subsequent version further expanded the user interface and available features. A fully customized GUI was created using Pygame, offering complex, reusable, and configurable parameters to suit user needs.

Additional Sensors: Integrated a motion sensor for enhanced detection capabilities.

Main Screen Features

Top Buttons: Settings, mode selector (Auto Home, Home Day and Night, Away, Out), and Wi-Fi status indicator.
Bottom Status Bar: Displays the status of sensors with color-coded indicators: open (red), closed (green), errors (yellow), no power (black), and unknown (gray).

Settings Screen Enhancements

Master Sound Settings: Adjust sound alerts for various conditions.
Mode Behavior Configuration: Customize how each mode responds to detected events.
Lifetime Statistics: View detailed analytics on system performance and usage.
Notification Management: Configure email and SMS alert behavior.
